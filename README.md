# IT Inventory

A simple Java program to keep track of IT equipment.

## Commands

- add - add a new item
- list - see all items
- search - find items by type or brand
- exit - save and quit

## Example

Comanda: add
Tip: router
Brand: Cisco
Descriere: birou principal
Produs adaugat.

Comanda: list
1. Tip: router | Brand: Cisco | Desc: birou principal
Total: 1 produse

Comanda: exit
Inventar salvat. Pa!
